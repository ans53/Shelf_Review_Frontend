  <p><strong>Shelf Review</strong> is a mobile app for book enthusiasts to share and explore reviews. It provides a simple, interactive, and mobile-friendly interface for discovering books and connecting with readers.</p>

  <h2>üéØ Aim & Objectives</h2>
  <ul>
    <li>Enable users to sign up, log in securely, and manage profiles.</li>
    <li>Share, view, and delete book reviews.</li>
    <li>Rate books using a 5-star system and upload related images.</li>
    <li>Engage with the community via likes and comments.</li>
    <li>Ensure the app is lightweight, responsive, and cross-platform.</li>
  </ul>

  <h2>‚öôÔ∏è Features</h2>
  <ul>
    <li>Home feed: Browse reviews with title, rating, caption, image, and author.</li>
    <li>Create reviews: Add new reviews with rating and optional image.</li>
    <li>Profile management: View posts, edit details, and delete reviews.</li>
    <li>Secure authentication using email and password.</li>
    <li>Community interaction via likes and comments.</li>
  </ul>
